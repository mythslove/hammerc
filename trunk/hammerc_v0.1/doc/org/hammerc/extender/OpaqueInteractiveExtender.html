Classpublic class OpaqueInteractiveExtender
InheritanceOpaqueInteractiveExtender Object

OpaqueInteractiveExtender 类提供支持位图不透明部分鼠标交互的功能扩展, 透明部分的鼠标交互事件会被忽略.

本类用来扩展指定对象并使其拥有特定的功能, 可采用复合的方式在需要该功能的类的子类中使用.
