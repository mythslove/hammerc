Classpublic class HotkeyObject
InheritanceHotkeyObject flash.events.EventDispatcher

HotkeyObject 类提供了侦听一组热键的功能.

设置侦听键盘按键时请使用 Keyborad 类的静态常量, 侦听鼠标按键时请使用本类的静态常量.

Parameters
interactiveObject:InteractiveObject — 需要侦听热键的交互对象.
 
name:String (default = null) — 热键对象的名称.
 
keys:Vector.<uint> (default = null) — 需要连续按下的按键列表.
 
intervalTime:int (default = 500) — 连续按下的有效间隔时间, 单位毫秒.
 
releaseKey:Boolean (default = false) — 触发热键完成事件时最后一个按键是按下 (false) 还是释放 (true). throws Error 参数 interactiveObject 为空时抛出该异常.
