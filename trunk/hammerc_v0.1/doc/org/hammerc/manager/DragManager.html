doDrag()method 
public function doDrag(dragInitiator:DisplayObject, dragData:DragData, mouseEvent:MouseEvent, dragImage:DisplayObject = null, offsetX:Number, offsetY:Number, imageAlpha:Number = 0.5):void

启动拖放操作.

Parameters

dragInitiator:DisplayObject — 指定启动拖动的显示对象.
 
dragData:DragData — 设置正在拖动的数据对象.
 
mouseEvent:MouseEvent — 设置与启动拖动相关的鼠标信息.
 
dragImage:DisplayObject (default = null) — 要拖动的图像, 默认使用黑边白底的矩形.
 
offsetX:Number (default = NaN) — 拖动图像的 x 轴偏移, 如指定偏移量则拖动代理对齐该偏移量, 否则使用鼠标按下的坐标.
 
offsetY:Number (default = NaN) — 拖动图像的 y 轴偏移, 如指定偏移量则拖动代理对齐该偏移量, 否则使用鼠标按下的坐标.
 
imageAlpha:Number (default = 0.5) — 拖动图像的透明度.
