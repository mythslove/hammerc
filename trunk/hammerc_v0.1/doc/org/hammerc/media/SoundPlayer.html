play()method 
public function play(position:Number = 0, loop:int = 1):void

开始播放声音, 设置的时间超出下载的能播放的时间或总时间时会自动设置到该声音的尾部.

Parameters

position:Number (default = 0) — 开始播放的时间秒数.
 
loop:int (default = 1) — 播放的次数, 0 为无限.


Throws
Error — 没有可用的播放通道时抛出该异常.
