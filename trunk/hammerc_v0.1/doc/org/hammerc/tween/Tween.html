Tween()Constructor
public function Tween(target:Object, duration:Number, variables:Object)

创建一个 Tween 对象.

Parameters
target:Object — 要进行缓动的对象, 传入空对象则本缓动对象将无效.
 
duration:Number — 缓动的持续时间, 负值按 0 处理.
 
variables:Object — 缓动的目标属性, 传入空对象则本缓动对象将无效, 可添加属性 paused:true 使缓动不会立即执行.
Method Detail
allFrom()method
public static function allFrom(targets:Array, duration:Number, variables:Object, gc:Boolean = true):Vector.<Tween>

缓动指定的多个对象到达该对象当前的状态.

Parameters

targets:Array — 要进行缓动的对象.
 
duration:Number — 缓动的持续时间.
 
variables:Object — 缓动的目标属性.
 
gc:Boolean (default = true) — 是否自动垃圾回收该缓动. 如果为 true 则该缓动的 gc 属性会被设置为 true, 即缓动结束后会调用自身的 destroy 方法实现垃圾回收. 这种情况下不要添加 paused:true 属性, 否则该缓动一创建好就会被回收, 创建默认不会被回收的缓动请直接用 new Tween(...) 的方法或者将本参数设置为 false.

Returns
Vector.<Tween> — 对应的缓动对象列表.

removeVariables()method 
public function removeVariables(variables:Object, restart:Boolean = false):void

移除一个已经存在的缓动目标属性, 该属性不会继续进行缓动.

Parameters

variables:Object — 要移除的目标属性. 如果移除一个属性并希望设置该属性的值请这样写 tween.removeVariables({x:100}); 这样属性 x 会在被移除的同时设置为 100, 如果希望该属性回到缓动前的值请这样写 tween.removeVariables({x:false}); 这样属性 x 会在被移除的同时设置回缓动前的值, 如果希望该属性停留在移除前的缓动值请这样写 tween.removeVariables({x:true}); 这样属性 x 会停止进行缓动.
 
restart:Boolean (default = false) — 是否要重新开始播放.
