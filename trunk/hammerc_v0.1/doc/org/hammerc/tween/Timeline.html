Method Detail
append()method
override public function append(tween:AbstractTweenCore, frame:Object = 0, align:int = 0):AbstractTweenCore

添加一个缓动对象到时间轴对象中.

    注意:
  • 添加后该缓动的运行起终时间点会被记录, 之后修改该缓动的持续时间会被本时间轴对象忽略.
  • 如果多个缓动对象要对同一个对象的属性进行缓动请配置其 overwrite 的属性为 TweenOverwrite.CONCURRENT 否则只会缓动最后添加的 Tween 中的属性.

Parameters

tween:AbstractTweenCore — 要添加的缓动或时间轴对象.
 
frame:Object (default = 0) — 指定的对象开始播放的帧标签或时间.
 
align:int (default = 0) — 该对象和位于同一帧的其它对象的播放方式. 第一个添加的 TimelineAlign.SEQUENCE 的缓动对象其开始时间为 0.

Returns
