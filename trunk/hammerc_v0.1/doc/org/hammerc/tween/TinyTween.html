Packageorg.hammerc.tween
Classpublic class TinyTween
InheritanceTinyTween Object

TinyTween 类实现了最基本的缓动同时体积却相当的小巧.
  • 去掉了复杂且不常用的功能, 在简单的交互应用中使用比 Tween 类更合适.
  • TinyTween 类为了达到超小的体积所以其独立于本包其它的缓动体系, 故不能使用 Timeline 对象操作本对象同时 TweenManager 无法对其进行控制请使用自身的静态方法来控制.
  • 使用本对象会增加大约 2.6kb 的大小.

from()method 
public static function from(target:Object, duration:Number, variables:Object, gc:Boolean = true):TinyTween

缓动指定对象到达该对象当前的状态.

Parameters

target:Object — 要进行缓动的对象.
 
duration:Number — 缓动的持续时间.
 
variables:Object — 缓动的目标属性.
 
gc:Boolean (default = true) — 是否自动垃圾回收该缓动. 如果为 true 则该缓动的 gc 属性会被设置为 true, 即缓动结束后会调用自身的 destroy 方法实现垃圾回收. 这种情况下不要添加 paused:true 属性, 否则该缓动一创建好就会被回收, 创建默认不会被回收的缓动请直接用 new Tween(...) 的方法或者将本参数设置为 false.

Returns
TinyTween — 对应的缓动对象.
