MULTI_LOAD_COMPLETEConstant
public static const MULTI_LOAD_COMPLETE:String = multiLoadComplete

每当所有下载项都下载完毕时触发.

此事件具有以下属性:

PropertyValue
item始终为 null.
numLoaded当前下载的项目索引.
numTotal需要下载的项目总数.
itemBytesLoaded始终为 0.
itemBytesTotal始终为 0.
networkSpeed始终为 0.
bubblesfalse
cancelablefalse
