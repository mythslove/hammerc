next()method 
public function next():*

返回迭代的下一个元素.

重复调用此方法直到 hasNext() 方法返回 false, 这将精确地一次性返回迭代器指向的集合中的所有元素.

Returns
* — 迭代的下一个元素.
remove()method 
public function remove():void

从迭代器指向的集合中移除迭代器返回的最后一个元素.

每次调用 next 只能调用一次此方法. 如果进行迭代时用调用此方法之外的其他方式修改了该迭代器所指向的集合，则迭代器的行为是不明确的.
