Classpublic class NewObjectPool
InheritanceNewObjectPool ObjectPool Object

NewObjectPool 类提供对象池的基本操作外对于注册过类型并添加到对象池中的对象在取出时进行一次还原, 还原后的对象与新生成的对象一致.
