dateFormat()method 
public static function dateFormat(date:Date, format:String = YYYY-MM-DD HH:NN:SS):String

将一个 Date 对象格式化为一个指定类型的字符串.

Parameters

date:Date — 需要转换的日期对象.
 
format:String (default = YYYY-MM-DD HH:NN:SS) — 转换后的格式.

支持的替换标签有:

  • 年: YYYY YY;
  • 月: MMMM MMM MM M (MM 用数字替换[如: 03], MMM 用字符替换[如：Mar]);
  • 日: DD D;
  • 星期: EEEE EEE EE (EE 为中文[如: 一], EEE 用字符替换[如: Wed]);
  • 上午下午: A (输出 am 或 pm);
  • 上午下午: C (输出 上午 或 下午);
  • 时: LL L (12小时制时间, 通常和 A 标签搭配);
  • 时: HH H;
  • 分: NN N;
  • 秒：SS S;

Returns
String — 对应该日期对象的格式化字符串.

Throws
ArgumentError — 参数 date 为空时抛出该异常.
