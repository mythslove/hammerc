sliceString()method 
public static function sliceString(string:String, startIndex:int = 0, endIndex:int = 0x7fffffff):String

截取指定字符串的一部分, 字符串中的中文及全角字符等将按长度为 2 来计算.

如果开始索引不足或结束索引不足则认为放弃该字符, 如 sliceString("hammerc基础类库v0.1", 8, 13) 会返回 "础类", 因为基字占用了 7 和 8 位, 而库字占用了 13 和 14 位.

Parameters

string:String — 需要截取的字符串.
 
startIndex:int (default = 0) — 开始截取索引.
 
endIndex:int (default = 0x7fffffff) — 结束截取索引.

Returns
String — 截取后的字符串.

substitute()method 
public static function substitute(string:String, ... rest):String

使用传入的各个参数替换指定的字符串内的 "{n}" 标记.

Parameters

string:String — 要在其中进行替换的字符串. 该字符串可包含 {n} 形式的特殊标记, 其中 n 为从零开始的索引, 它将被该索引处的其他参数 (如果指定) 替换.
 
... rest — 可在 string 参数中的每个 {n} 位置被替换的其他参数, 其中 n 是一个对指定值数组的整数索引值 (从 0 开始). 如果第一个参数是一个数组, 则该数组将用作参数列表.

Returns
String — 使用指定的各个参数替换了所有 {n} 标记的新字符串.
