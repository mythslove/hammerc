getObjectsUnderPoint()method 
public static function getObjectsUnderPoint(target:DisplayObject, point:Point, displayList:Vector.<DisplayObject>):void

获取指定显示对象中特定点下方的所有显示对象.

由于 FlashPlayer 自带的 DisplayObjectContainer.getObjectsUnderPoint() 方法检测的是上一帧渲染时的显示列表的状态, 所以这里实现能获取到实时显示列表的方法.

Parameters

target:DisplayObject — 需要获取指定点下方显示列表的显示对象.
 
point:Point — 舞台坐标空间中指定的点.
 
displayList:Vector.<DisplayObject> — 存放指定点下方的所有显示对象的列表, 按嵌套深度从低到高排序, 如果存在显示对象则第一个对象为指定的显示对象.
