take()method 
public function take(filter:Function = null, ... args):*

获取一个闲置的对象.

若对象池已空且注册过类型则创建新对象返回.

Parameters

filter:Function (default = null) — 对象池获取对象的过滤方法. 接收一个当前遍历的对象池中对象的 类型参数, 返回布尔值指定是否取出当前的这个对象.
 
... args — 如果对象池中的对象已空需要创建新对象时, 传递给构造函数的参数.

Returns
* — 一个闲置的对象, 如果对象池中已经没有对象则返回新对象或没有注册类型则返回 null.
