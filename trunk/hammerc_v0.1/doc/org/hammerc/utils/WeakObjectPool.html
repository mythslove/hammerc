Packageorg.hammerc.utils
Classpublic class WeakObjectPool
InheritanceWeakObjectPool Object
Implements IObjectPool

WeakObjectPool 类定义了弱引用的对象池对象.

如果添加到本对象池的对象仅被本对象池对象引用则该对象会在下一次垃圾回收时被回收.
