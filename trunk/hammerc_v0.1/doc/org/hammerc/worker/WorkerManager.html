Packageorg.hammerc.worker
Classpublic class WorkerManager
InheritanceWorkerManager Object

WorkerManager 类提供和 WorkerDomain 类类似的方法, 同时封装了线程的连接.

注意静态方法 createWorker 创建一个基于主线程的子线程, 同时会创建两个消息通道连接主线程和子线程.

Method Detail
createWorker()method
public static function createWorker(swf:ByteArray, giveAppPrivileges:Boolean = false):WorkerData

基于某个 swf 的字节创建一个新的 Worker 实例并和主线程建立消息通道.

Parameters

swf:ByteArray — 包含有效 swf 的字节的 ByteArray.
 
giveAppPrivileges:Boolean (default = false) — 指示在 AIR 中是否应当向 worker 授予应用程序沙箱权限.

Returns
WorkerData — 成功时返回新创建的 WorkerData 对象. 返回值为 null 时表示无法创建 worker, 因为当前上下文不支持并发或者创建新 worker 将超出实现限制.
