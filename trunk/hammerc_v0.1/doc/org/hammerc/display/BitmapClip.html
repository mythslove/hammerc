InheritanceBitmapClip flash.display.Sprite
Subclasses OpaqueInteractiveBitmapClip

BitmapClip 类用于优化矢量图组成的影片剪辑.

所有矢量图都会被转换为位图进行显示, 故本对象在拉伸旋转的情况下会出现锯齿现象.

addFrameScript()method 
public function addFrameScript(frame:Object, script:int, gotoFrame:Object = null):void

向指定帧对象添加脚本.

  • 仅支持四种基本的播放跳转控制脚本.
  • 被设置的帧如果已经存在脚本则新设置的脚本会替换旧脚本.
  • 帧脚本全局通用, 只要向任一实例添加帧脚本, 指向同样源影片剪辑的所有位图剪辑都将获得同样的帧脚本.

Parameters

frame:Object — 要添加脚本的帧对象.
 
script:int — 要添加的脚本类型, 设置为 BitmapFrame.NO_SCRIPT 会移除指定帧的脚本.
 
gotoFrame:Object (default = null) — 如果为跳转类型的脚本本参数指定其会跳转到的帧对象.
